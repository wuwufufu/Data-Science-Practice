# AttnRes-LoRA / Yelp VLM Experiment Code Package

This archive contains the complete experimental code used for the Yelp VLM PEFT experiments, including vanilla LoRA, multi-expert/TMR-LoRA variants, depth-attention TMR, and the final Attention Residual LoRA implementation.

## Included

- `LIFT-Yelp-code/`: full source code tree, scripts, configs, dataset loaders, CLIP utilities, tests, and documentation.
- `LIFT-Yelp-code/train_vlm_lora.py`: main training/evaluation entry for VLM LoRA, TMR-LoRA, depth-attention TMR, and AttnRes-LoRA experiments.
- `LIFT-Yelp-code/evaluate_vlm_direct.py`: direct VLM evaluation entry.
- `LIFT-Yelp-code/build_vlm_lora_dataset.py`: dataset construction for VLM fine-tuning.
- `LIFT-Yelp-code/vlm_tmr_lora.py`: VLM adapter implementations, including TMR/Depth-TMR/AttnRes-style modules.
- `LIFT-Yelp-code/models/`: baseline PEFT and TMR modules for the original CLIP-side experiments.
- `LIFT-Yelp-code/configs/` and `LIFT-Yelp-code/scripts/`: original experiment configs and batch scripts.
- `LIFT-Yelp-code/datasets/`: dataset loader code and split/list metadata used by the experiments.
- `LIFT-Yelp-code/outputs/`: only lightweight experiment records are included, such as `config.yaml`, `metrics.json`, `train.log`, adapter config JSON files, trainer state JSON files, README files, and `loss_curve.png`.

## Excluded

Large generated artifacts are intentionally excluded from this code package:

- trained adapter weights: `*.pt`, `*.pth`, `*.bin`, `*.safetensors`
- optimizer/scheduler states and checkpoint tensors
- Python caches: `__pycache__/`, `*.pyc`
- external VLM backbone weights and downloaded model files
- raw image data archives

The excluded files are not source code. They can be regenerated by running the scripts with the same configs, assuming the local VLM backbone and Yelp image/text data are available.

## Main Reproduction Pointers

Typical commands are centered around:

```bash
cd LIFT-Yelp-code
python3 evaluate_vlm_direct.py --help
python3 build_vlm_lora_dataset.py --help
python3 train_vlm_lora.py --help
```

The final method is the AttnRes-LoRA / attention-residual variant. The key implementation lives in `vlm_tmr_lora.py` and is exercised through `train_vlm_lora.py` with the corresponding adapter method arguments and configs recorded under `outputs/vlm_attnres_lora/`.
